## Installation

Install _FRONTEND_ with npm

```bash
  cd front
  npm install
  npm run dev
```

Install _BACKEND_ with npm

```bash
  cd back
  pipenv install
  pipenv run start
```

## Changes in Models

```bash
  pipenv run migrate
  pipenv run upgrade
```
